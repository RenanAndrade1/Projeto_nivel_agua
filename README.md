# Programa de controle de nível de água

esse programa tem o objetivo de mostrar os diferentes níveis de quantidade de água de um reservatório.

o programa utiliza a biblioteca colorama para exibir mensagens de cores personalizadas.

## Atenção ⚠️
Ao final do sistema, as cores do terminal voltam ao normal
